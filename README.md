## 🚀 העלאה לנטליפיי (Netlify)

### אופציה 1️⃣: Drag & Drop (הכי פשוט!)

1. **היכנס לנטליפיי:**
   - גש ל-[netlify.com](https://netlify.com)
   - התחבר או הירשם (חינם!)

2. **העלה את התיקייה:**
   - בדף הראשי לחץ על "Add new site" → "Deploy manually"
   - גרור את **כל התיקייה `agent-guy`** לאזור ההעלאה
   - **חשוב:** וודא שכל הקבצים נכללים, במיוחד הסרטונים!

3. **המתן לפריסה:**
   - נטליפיי יעלה את כל הקבצים (זה יכול לקחת 2-5 דקות)
   - תקבל URL זמני כמו: `https://random-name-123456.netlify.app`

4. **שנה שם (אופציונלי):**
   - לחץ על "Site settings" → "Change site name"
   - בחר שם כמו: `agent-guy-malka` או `guy-crownbet`

---

### אופציה 2️⃣: דרך GitHub (מקצועי יותר)

1. **צור repository ב-GitHub:**
   ```bash
   cd agent-guy
   git init
   git add .
   git commit -m "Initial commit - Agent Guy landing page"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/agent-guy.git
   git push -u origin main
   ```

2. **חבר לנטליפיי:**
   - בנטליפיי: "Add new site" → "Import an existing project"
   - בחר GitHub → בחר את ה-repository
   - הגדרות:
     - Build command: (השאר ריק)
     - Publish directory: `.`
   - לחץ "Deploy site"

3. **יתרונות:**
   - עדכונים אוטומטיים בכל push
   - גרסאות שמורות
   - Rollback קל
